IEEE-754 Floating Point Converter

Precision:
Byte order display:
OK layout: 1 / 8 / 23

Input / Output

Typing here converts to IEEE-754 and updates Hex/Binary immediately.
Interpreted as raw IEEE-754 bits (big-endian bit order). Accepts optional 0x.
You may add spaces/underscores; length will be left-padded with zeros if shorter.
Toggle any bit checkbox to update Hex/Binary/Decimal and decoded values.

Special cases: E=all 1 & M=0 → ±Infinity; E=all 1 & M≠0 → NaN; E=all 0 & M=0 → ±0; E=all 0 & M≠0 → Subnormal.

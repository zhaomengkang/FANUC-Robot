程序：gi32_recv、go32_send
释放版本日期：2025年9月20日

作者：赵梦康  964210817@qq.com

程序的安装：
1.  将文件拷贝至U盘根目录，文件树如下
──────────────────────────────────
│  install.cm
│  readmech.txt
│  uninstall.cm
│
├─7DC文件夹
│      ap.com
│      argdispchmklb.dt
│      argdispegmklb.dt
│      del.com
│      gi32_recv.pc
│      go32_send.pc
│
└─7DF文件夹
        ap.com
        argdispchmklb.dt
        argdispegmklb.dt
        del.com
        gi32_recv.pc
        go32_send.pc
──────────────────────────────────

2. 示教器文件菜单内，光标选中"install.cm"，按下ENTER按键，提示是否执行？选择是
3. 执行完毕提示"Install done,Please Cold Start" 表示完成
4. 执行冷启动
